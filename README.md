# excalidraw-scripts
A collection of Excalidraw scripts to directly create Markdown files and automatically generate backlinks from arrow placement.

## Generate Markdown file scripts

### Create markdown note (id)
Creates a new Markdown note in the current folder using a timestamp as the filename, then automatically adds it to the canvas.
### Create markdown note (input)
Prompts you to name a new Markdown note, saves it in the current folder, and adds it to the canvas.
### Delete markdown note
Delete the mardown note the note currently selected on the canvas.

## Automatic backlinking scripts

### Generate backlinks from arrows (current drawing)
Automatically creates backlink properties for any notes connected by arrows in the current canvas.
### Remove backlinks generated from arrows (current drawing)
Removes all backlink properties generated by arrows within the current canvas.
### Find orphaned arrows
Identifies any arrows on the canvas that are not attached to a note.
### Remove orphaned backlinks generated from arrows (vault)
Scans the entire vault and removes backlinks pointing to non-existent notes (useful for cleanup after deleting files).
